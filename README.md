## 🧠 Core Philosophy

EverBond Wealth is built around three distinct life stages, ensuring that the platform evolves in tandem with the user's relationship journey:

*   **Single**: Focuses on **Personal financial planning**, enabling users to gain solo financial autonomy, structure income targets, build savings, and model their own future index targets.
*   **Committed**: Transitions to **Shared future planning**, showing joint timelines, managing downpayments collaboratively, split savings estimators, and connection checks.
*   **Married**: Orchestrates **Family wealth planning**, operating unified ledgers, multi-generational college indexes, child trusts, and combined retirement run-rates.

## 🆔 EverBond ID System

Every user receives a unique **EverBond ID** upon profile creation:

```
Example: EB-A7K92X
```

The EverBond ID acts as the core identity layer for future partner synchronization and collaborative financial planning. It enables users to securely associate profiles, merge financial views, and plan future assets collaboratively.

## 🔄 Partner Connection Flow

Hydrating your dashboard to unlock shared capabilities follows an elegant connection handshake:

```
  1. Create Profile   ──→   2. Receive EverBond ID   ──→   3. Connect Partner
                                                                  │
  6. Send Request     ←──   5. Add Anniversary       ←──   4. Enter Partner ID
        │
        ▼
  7. Shared Journey Activated!
```

1.  **Create Profile**: Complete onboarding steps.
2.  **Receive EverBond ID**: Get your unique copyable identity key.
3.  **Connect Partner**: Open the Connection Center.
4.  **Enter Partner EverBond ID**: Paste your partner's unique client key.
5.  **Add Anniversary Date**: Record your milestone date.
6.  **Send Request**: Initialize connection parameters.
7.  **Shared Journey Activated**: Instantly transitions workspaces into synced parallel modes!
