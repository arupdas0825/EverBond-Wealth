## ✨ Features

### 💎 Core Platform
| Feature | Description |
|---------|-------------|
| **Excel Brain** | Exact formulas from `Investment_Planner.xlsx` — no approximations |
| **Couple-First** | Designed for two — combined salary engine, shared goals |
| **Zero Auth** | No login, no cloud — your data lives locally on your device |
| **PWA Ready** | Install to home screen, works offline, app-like on mobile |
| **Multi-Currency** | INR, USD, EUR, GBP, CHF, CAD, SGD, AED |

### 📊 Financial Modules
| Module | Excel Section |
|--------|--------------|
| **Income & Mode** | Section 1 — Salary Budget & Allocation Mode |
| **Investment Allocation** | Section 2 — Investment Split (Equity/Debt/Commodities/Crypto) |
| **Equity Engine** | Section 3 — Equity Breakdown (Large/Mid/Small Cap) |
| **Debt Engine** | Section 4 — Debt Breakdown (Liquid/Short TMF/Target Maturity) |
| **Commodities Engine** | Section 5 — Gold & Silver Split |
| **Goal Funding Engine** | Section 6 — Goal Split (Child/Retirement/Home/Vacation) |
| **SIP Simulation** | FV = PMT × [((1+r)ⁿ − 1) / r] × (1+r) |

- **Safe area aware** — no content behind notch or home indicator
- **Reset Platform** in mobile "More" drawer
- **Fully touch optimized** — no small tap targets

## 🧮 Financial Logic

> **The Excel file is the single source of truth.** Every calculation in EverBond Wealth is a faithful JavaScript translation of the formulas in `Investment_Planner.xlsx`.

### Excel → Code Mapping

```
Excel: Monthly Salary × 0.55 → Essentials
Code:  salary * presets.needs  → budget.needs

Excel: Investments × 0.60 → Equity (Balanced mode)
Code:  investments * presets.equity → investmentSplit.equity

Excel: Equity × 0.60 → Large Cap
Code:  equity * presets.equityLarge → equityBreakdown.largeCap

Excel: FV = PMT × [((1+r)^n − 1)/r] × (1+r)
Code:  monthlyAmount * ((Math.pow(1+r,n)-1)/r) * (1+r)
```

### Allocation Presets (from `Presets` sheet)

| Mode | Needs | Emergency | Invest | Equity | Debt | Commodities | Crypto |
|------|-------|-----------|--------|--------|------|-------------|--------|
| **Conservative** | 60% | 10% | 30% | 40% | 50% | 8% | 2% |
| **Balanced** | 55% | 10% | 35% | 60% | 30% | 8% | 2% |
| **Aggressive** | 50% | 10% | 40% | 70% | 20% | 8% | 2% |

## 🚀 Quick Start

### Prerequisites

- **Node.js** `≥ 18.0.0`
- **npm** `≥ 9.0.0`

### Local Development

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/everbond-wealth.git
cd everbond-wealth

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev

# 4. Open in browser
# → http://localhost:5173
```

### Production Build

```bash
npm run build        # Builds to ./dist
npm run preview      # Preview production build locally
```

## ☁️ Deployment

### Vercel (Recommended — One Click)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/everbond-wealth)

Or manually:

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy from project root
vercel --prod
```

### GitHub Actions CI/CD Setup

The pipeline at `.github/workflows/deploy.yml` provides:

1. **Quality Gate** — Lint + Build check on every push and PR
2. **Production Deploy** — Auto-deploys to Vercel on push to `main`
3. **Preview Deploy** — Auto-deploys preview URL on every pull request

#### Required GitHub Secrets

Go to **Settings → Secrets and variables → Actions** and add:

| Secret | Where to find |
|--------|--------------|
| `VERCEL_TOKEN` | [Vercel Dashboard → Settings → Tokens](https://vercel.com/account/tokens) |
| `VERCEL_ORG_ID` | `.vercel/project.json` after running `vercel` once |
| `VERCEL_PROJECT_ID` | `.vercel/project.json` after running `vercel` once |

```bash
# Run this once locally to generate .vercel/project.json
vercel

# Then copy the IDs to GitHub Secrets
cat .vercel/project.json
```
