# EverBond Wealth
> A premium couple-focused financial planning and wealth management platform.

EverBond Wealth is a modern fintech-inspired web platform designed to help couples and families plan their financial future together in a beautiful, emotionally warm, and visually premium experience.

The project combines:
- Couple financial planning
- Wealth management
- Goal tracking
- Investment allocation
- Future wealth simulation
- Multi-currency support
- Premium UI/UX

This project is inspired by a professional Excel-based investment planning system and aims to transform complex financial planning into a smooth, elegant, and emotionally engaging experience.

---

## ✨ Current Features

### 💖 Couple-Friendly Onboarding
- Beautiful welcome experience
- Partner name setup
- Local device personalization
- Region selection support
- Currency selection
- LocalStorage-based onboarding persistence

### 🌍 Multi-Currency Support
**Supported currencies:** INR, USD, EUR, GBP, CHF, CAD, SGD, AED
- Dynamic currency formatting
- Intl.NumberFormat integration
- Currency-aware dashboard visuals
- Currency-specific UI formatting

### 📊 Premium Dashboard
- Financial overview cards
- Animated dashboard widgets
- Savings visualization
- Investment tracking
- Financial health section
- Responsive fintech UI

### 💰 Income Planning System
- Separate partner income inputs
- Combined monthly income calculation
- Financial allocation foundation
- Couple-centric financial flow

### 📈 Investment Allocation UI
**Supports:** Equity, Debt, Gold, Crypto, Emergency Fund
**Risk profiles:** Conservative, Balanced, Aggressive

### 🎯 Goal Planning
Users can plan for: Home purchase, Emergency fund, Child education, Germany study plan, Retirement, Vacation, Car purchase

### 🎬 Premium UI/UX
- Glassmorphism design
- Smooth animations
- Framer Motion transitions
- Premium gradients
- Responsive layouts
- Modern sidebar navigation
- Elegant chart sections
